## 프로젝트 기능
---
객체 지향 설계를 적용해 상품 관리 시스템을 프로그래밍 하였습니다.

- [x] 상품 목록 출력
  - [x] JAVA 프로그램을 실행하면 여러 전자제품을 출력합니다.
  - [x] 제시된 상품 중 입력받은 숫자에 따라 다른 로직을 실행하는 코드를 작성합니다.


- [x] Product 클래스 생성하기
  - [x] 개별 상품을 관리하는 클래스입니다.
  - [x] *상품명*, *가격*, *설명*, *재고 수량* 필드를 갖습니다.

---
객체 지향 설계를 적용해 순서 제어를 클래스로 관리하였습니다.

- [x] CommerceSystem 클래스 생성하기

  - [x] 커머스 플랫폼의 상품을 관리하고 사용자 입력을 처리하는 클래스입니다.
  - [x] 입력과 반복문 로직을 start 함수를 만들어 관리합니다.
  - [x] List<Category>를 관리합니다.
  - [x] CommerceSystem 객체를 생성하고 사용하는 main 함수에서 객체를 생성할 때 값을 넘겨줍니다.

객체 지향 설계를 적용해 상품 카테고리와 고객을 클래스 기반으로 관리하였습니다.

- [x] Category 클래스 생성하기
  - [x] Product 클래스를 관리하는 클래스입니다.
  - [x] 전자제품, 의류, 식품 등 각 카테고리 내 여러 *Product*를 만들어줍니다.

  - [x] 여러 상품들을 포함하는 상위 개념 '전자제품' 같은 *카테고리 이름* 필드를 갖습니다.
  - [x] 카테고리 이름을 반환하는 메서드가 구현되었습니다.

- [x] Customer 클래스 생성하기
  - [x] 고객 정보를 관리하는 클래스입니다.
  - [x] *고객명*, *이메일*, *등급* 필드를 갖습니다.

### Lv 1. 장바구니 및 주문하기 기능을 추가하기

- [x] 장바구니 생성 및 관리 기능
  - [x] 사용자가 선택한 상품을 장바구니에 추가할 수 있는 기능을 제공합니다.
  - [ ] 장바구니는 상품명, 수량, 가격 정보를 저장하며, 항목을 동적으로 추가 및 조회할 수 있어야 합니다.
  - [x] 사용자가 잘못된 선택을 했을 경우 예외를 처리합니다. (예: 유효하지 않은 상품 번호 입력)

- [x] 재고 관리 시스템
  - [x] 상품을 장바구니에 담을 때 재고를 확인하고, 재고가 부족할 경우 경고 메시지를 출력합니다.
  - [x] 주문 완료 시 해당 상품의 재고를 차감합니다.

- [x] 장바구니 출력 및 금액 계산
  - [x] 사용자가 주문을 시도하기 전에, 장바구니에 담긴 모든 상품과 총 금액을 출력합니다.
  - [x] 출력 예시
    - [x] 각 상품의 이름, 가격, 수량
    - [x] 총 금액 합계

- [x] 장바구니 담기 기능
  - [x] 상품을 선택하면 장바구니에 추가할 지 물어보고, 입력값에 따라 "추가", "취소" 처리합니다.
  - [x] 장바구니에 담은 목록을 출력합니다.

- [x] 주문 기능
  - [x] 장바구니에 담긴 모든 항목을 출력합니다.
  - [x] 합산하여 총 금액을 계산하고, "주문하기"를 누르면 장바구니를 초기화하고 재고를 차감합니다.
