#### 프로젝트 기능
---
객체 지향 설계를 적용해 상품 관리 시스템을 프로그래밍 하였습니다.

- [x] 상품 목록 출력
  - [x] JAVA 프로그램을 실행하면 여러 전자제품을 출력합니다.
  - [x] 제시된 상품 중 입력받은 숫자에 따라 다른 로직을 실행하는 코드를 작성합니다.


- [x] Product 클래스 생성하기
  - [x] 개별 상품을 관리하는 클래스입니다.
  - [x] *상품명*, *가격*, *설명*, *재고 수량* 필드를 갖습니다.

---
객체 지향 설계를 적용해 순서 제어를 클래스로 관리하였습니다.

- [x] CommeerceSystem 클래스 생성하기
  - [x] 커머스 플랫폼의 상품을 관리하고 사용자 입력을 처리하는 클래스입니다.
  - [x] 입력과 반복문 로직을 start 함수를 만들어 관리합니다.
  - [x] List<Category>를 관리합니다.
  - [x] CommerceSystem 객체를 생성하고 사용하는 main 함수에서 객체를 생성할 때 값을 넘겨줍니다.

---
객체 지향 설계를 적용해 상품 카테고리와 고객을 클래스 기반으로 관리하기

- [x] Category 클래스 생성하기
  - [x] Product 클래스를 관리하는 클래스입니다.
  - [x] 전자제품, 의류, 식품 등 각 카테고리 내 여러 *Product*를 만들어줍니다.
  - [x] List<Product>를 관리합니다.
  - [x] 여러 상품들을 포함하는 상위 개념 '전자제품' 같은 *카테고리 이름* 필드를 갖습니다.
  - [x] 카테고리 이름을 반환하는 메서드가 구현되었습니다.

- [x] Customer 클래스 생성하기
  - [x] 고객 정보를 관리하는 클래스입니다.
  - [x] *고객명*, *이메일*, *등급* 필드를 갖습니다.
